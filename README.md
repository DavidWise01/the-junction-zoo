# THE JUNCTION ZOO · JZ2

> A sphere of **UD0** — the ROOT0 universe / biosphere. Domain: **PSĒPHOS**.

THE JUNCTION ZOO (field guide II) — almost the entire electronics shelf from one trick, p against n, wired into different shapes: 22 doped semiconductor devices in 5 families with LIVE carrier-drift diagrams (the moving dots are real carriers, not decoration). DIODES (p-n, Zener, Schottky, PIN, tunnel/Esaki, varactor) · TRANSISTORS (BJT, MOSFET, JFET, IGBT, HEMT, thyristor) · OPTOELECTRONICS (LED, laser diode, photodiode, solar cell, image sensor) · SENSE (Hall, Gunn, IMPATT) · MEMORY (flash floating-gate, DRAM 1T1C). Each card: a cross-section + the real dopants + what it does. David's own work, render-not-invent, honest (schematic geometry flagged). Companion to [[the-doped-crystal]] — the device layer above the substrate.
